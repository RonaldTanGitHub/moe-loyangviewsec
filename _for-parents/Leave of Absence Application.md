# **Leave of Absence Application** 

Parents / Guardians, if there is a need for your child / ward to be absent from school for a prolonged period due to some unforeseen circumstances or events, do apply for an official leave of absence from the school. 

You may do so by using the following FormSG link ->
    [https://form.gov.sg/60c6e49ce8d356001135a4a2](https://form.gov.sg/60c6e49ce8d356001135a4a2)
		
Please contact the school for assistance if you have difficulty doing so.
